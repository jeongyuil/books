# 작업 1: 표지 문구

---

## A. 부제 (한글) — 3가지 후보

1. **콘텐츠 플랫폼의 시간 전쟁에서 이기는 데이터 전략**
   - 프롤로그의 핵심 메타포인 '시간 탈취'와 '전장'을 살리면서, 데이터 전략이라는 실무적 가치를 명시

2. **넷플릭스, 디즈니+를 움직이는 데이터 성장의 기술**
   - 타겟 독자가 즉시 반응할 플랫폼명을 노출하고, '기술'이라는 단어로 실용서임을 강조

3. **사용자의 선택을 설계하는 플랫폼 데이터 실전 가이드**
   - 프롤로그의 '사용자의 선택을 이해하는 기술'에서 한 걸음 더 나아가, 독자가 실무에 바로 적용할 수 있다는 기대를 생성

---

## B. 띠지 카피 — 3가지 후보

### 질문형
> **저녁 9시 47분, 당신의 손가락은 누구의 설계 위에 있는가?**

- 프롤로그 오프닝의 시간(9시 47분)과 '손가락의 움직임을 읽는다'는 핵심 장면을 그대로 활용. 서점에서 책을 집어든 독자에게 "나도 모르게 조종당하고 있는 건 아닌가"라는 호기심을 자극.

### 선언형
> **감으로 운영하는 시대는 끝났다. 이제 데이터가 성장을 설계한다.**

- 프롤로그 마지막 메시지("더 이상 감으로 운영하는 플랫폼을 만들지 않을 것이다")를 압축. 실무자의 전환 의지를 자극하는 선언.

### 숫자/구체성 강조형
> **Disney+ APAC, EA 글로벌 — 14년간 플랫폼 내부에서 본 데이터 성장의 모든 것**

- 저자의 경력을 구체적 숫자와 브랜드로 제시해 신뢰도를 높임. "내부자가 쓴 책"이라는 차별점을 한 줄에 압축.

---

## C. 뒷표지 소개문 (약 180자)

# 작업 2: 디자이너 브리프

---

## 1. 무드/톤 — 3가지 키워드

| 키워드 | 설명 |
|--------|------|
| **지적인 긴장감 (Intelligent Tension)** | 프롤로그가 '보이지 않는 전쟁'과 '몇 초의 망설임'을 다루듯, 표지도 차분하면서 긴장을 품어야 한다. 소리치지 않지만 시선을 잡아두는 톤. |
| **절제된 내부자 (Restrained Insider)** | 화려한 데이터 시각화가 아니라, 내부자만 아는 조용한 확신이 느껴지는 톤. 비밀을 알고 있는 사람의 표정 같은 느낌. |
| **한국적 현실감 (Grounded Realism)** | 실리콘밸리 판타지가 아닌, 서울 강동구 거실과 브루클린 코워킹 스페이스가 공존하는 구체적 현실의 질감. |

---

## 2. 참고 이미지 방향

### 추천하는 방향
- **어두운 배경 위에 빛나는 UI 요소의 추상적 잔상**: 스마트폰이나 TV 화면에서 나오는 빛이 어둠 속에서 번지는 듯한 이미지. 콘텐츠 썸네일이 격자로 배열되어 있지만 약간 흐릿하게 처리되어, "무엇을 볼지 고르는 그 순간"의 느낌을 전달.
- **데이터 흐름의 최소화된 표현**: 점과 선으로 이루어진 절제된 네트워크 그래프 또는 사용자 행동 흐름(funnel)의 추상적 패턴. 화려한 대시보드가 아니라, 한두 개의 곡선이나 노드로 데이터의 존재를 암시.
- **손가락 하나의 실루엣**: 프롤로그에서 반복되는 "손가락의 움직임을 읽는다"는 모티프를 살려, 리모컨이나 화면을 누르려는 손가락 하나의 실루엣을 최소한으로 배치.

### 구체적 시각 묘사
밤 9시의 거실을 연상시키는 어두운 톤 위에, 화면에서 새어 나오는 푸른빛/보라빛 잔광이 부드럽게 퍼진다. 중앙 하단 또는 한쪽에 데이터 흐름을 암시하는 미세한 선/점 패턴이 있다. 전체적으로 "조용하지만 무언가 움직이고 있다"는 인상.

## 3. 색상 방향 — 3가지 조합 제안

### 조합 1: 미드나이트 블루 + 일렉트릭 바이올렛 (추천)
- **주조색**: 깊은 남색(#0A1628) — 밤의 거실, 스크린 꺼진 어둠
- **강조색**: 전기 보라(#7B2FBE) — 스트리밍 플랫폼의 빛, 데이터의 움직임
- **텍스트**: 밝은 백색(#F0F0F0) 또는 연한 라벤더(#C4B5E0)
- **이유**: 스트리밍 서비스의 밤 시간대 시청 경험을 연상시키면서, 기술서적의 고급감을 동시에 전달. Netflix/Disney+의 어두운 UI 톤과도 자연스럽게 연결.

### 조합 2: 차콜 블랙 + 시그널 레드
- **주조색**: 차콜 블랙(#1A1A1A)
- **강조색**: 깊은 빨강(#C62828) — 긴장감, 전쟁의 메타포
- **텍스트**: 순백(#FFFFFF)
- **이유**: 프롤로그의 '보이지 않는 전투'라는 긴장감을 시각적으로 극대화. 서점 진열대에서 강한 대비로 시선을 잡아끌 수 있음.

### 조합 3: 다크 네이비 + 골드
- **주조색**: 다크 네이비(#0D1B2A)
- **강조색**: 차분한 금색(#C9A94E) — 내부자의 지식, 가치 있는 정보
- **텍스트**: 오프화이트(#EAEAEA)
- **이유**: "비밀을 아는 사람"의 프리미엄한 느낌을 전달. 비즈니스 서적의 권위감 있으면서도 콘텐츠 산업의 현대적 감각 유지.

## 4. 타이포그래피 방향

### 한글 제목
- **방향**: 고딕 계열이되, 완전한 기하학적 산세리프가 아닌 약간의 붓 터치 또는 인간적 무게감이 있는 서체
- **참고 서체 느낌**: Noto Sans KR Bold/Black 또는 Pretendard ExtraBold 수준의 무게감, 또는 산돌 고딕Neo 계열의 단단한 고딕
- **이유**: 메인 제목이 문장형("그들은 당신이 뭘 볼지 이미 알고 있다")이므로, 명조 계열은 소설처럼 보일 위험이 있음. 고딕으로 가되, 기계적이지 않은 따뜻한 무게감이 필요
- **자간**: 약간 좁게. 문장이 길기 때문에 자간을 조여서 한 덩어리로 읽히게 처리
- **배치**: 2~3줄로 나누되, "이미 알고 있다"가 마지막 줄에서 강조되는 레이아웃

### 영문 제목/부제
- **방향**: 기하학적 산세리프. 깔끔하고 현대적, 테크 기업의 프레젠테이션 느낌
- **참고 서체 느낌**: Inter, DM Sans, 또는 Gilroy 계열
- **크기**: 한글 제목 대비 50~60% 크기로 부제 위치에 배치

### 저자명
- **방향**: 한글 제목과 같은 서체 패밀리의 Regular 또는 Light 웨이트
- **한/영 병기**: "정유일 Elliott Jeong" — 한글을 먼저, 영문을 작게

## 5. 피해야 할 것

| 피해야 할 요소 | 이유 |
|----------------|------|
| **화려한 데이터 대시보드 이미지** | 데이터 엔지니어링 기술서처럼 보임. 이 책은 '전략서'이자 '에세이'에 가까움 |
| **눈동자/감시 카메라/빅브라더 이미지** | 제목의 "이미 알고 있다"에서 디스토피아적 감시를 연상시키면 톤이 무너짐. 이 책은 공포가 아니라 성장 전략을 다룸 |
| **넷플릭스/디즈니+ 로고 직접 사용** | 저작권 문제 및 특정 플랫폼 홍보서로 오해될 수 있음 |
| **밝은 파스텔 톤, 일러스트 캐릭터** | 자기계발서나 입문서의 톤으로 빠짐. 타겟 독자는 현직 실무자이므로 가벼워 보이면 안 됨 |
| **과도하게 복잡한 인포그래픽** | 표지에서 정보를 전달하려 하면 안 됨. 호기심과 분위기만 전달해야 함 |
| **전형적인 IT/경영서의 파란색+흰색 조합** | 차별화 실패. 서점 경영/IT 코너에서 묻힘 |
| **사람 얼굴 사진** | 저자 사진은 날개에만. 표지에 인물 사진을 넣으면 자기계발/에세이 톤이 됨 |
